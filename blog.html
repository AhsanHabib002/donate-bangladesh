What is the Document Object Model (DOM)?

The Document Object Model (DOM) provides a programming interface for web content. It represents the page, allowing applications to modify the document's structure, design, and content.

How do you select an element from the DOM?

There are common 5ways to select an element from the dom. 1. getElementByID, 2.getElementsByClassName, 3.getElementsByTagName, 4. querySelector, 5. querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is the technique of employing event propagation to handle events at a higher level in the DOM than the element where the event originated. It enables us to link a single event listener to items that exist now or in the future.
